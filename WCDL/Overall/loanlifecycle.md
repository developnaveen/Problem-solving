# Loan life cycle 

# Loan flow
Customer → Loan Application → Credit Assessment → Sanction
→ Disbursement → Utilization → Interest Calculation
→ Repayment → Closure / Renewal


## table 
Banking terms you MUST understand 
Term	What it means for code
Sanction	Max allowed value
Disbursement	Money release event
Outstanding	Current balance
Utilization	Used amount
Tenure	End date calculation
DP (Drawing Power)	Max withdrawal
NPA	Status flag
Moratorium	Skip EMI logic
Penal Interest	Extra rate
